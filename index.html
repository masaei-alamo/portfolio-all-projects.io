Applied Data Analysis – Final Python Project

A comprehensive individual project that combines real-world datasets and advanced data analytics using Python. The project integrates data from two domains: global COVID-19 statistics and global coffee production, cleaning and merging them into one dataset to derive insights using visual and statistical tools.

  • Step A – Data Source Selection: Chose two real-world CSV files: COVID-19 global dataset and global coffee production dataset.
  • Step B – Data Quality Check: Handled missing values, reviewed data types, removed duplicates, and ensured consistency before analysis.
  • Step C – Data Cleaning & Transformation: Merged datasets using outer join, standardized column names, removed outliers, and formatted time/date fields appropriately.
  • Step D – Visual Data Analysis: Produced a variety of advanced charts using Plotly and Seaborn: bar charts, bubble charts, sunburst, treemap, Sankey diagram, heatmap, and interactive choropleth maps.
  • Step E – Statistical Analysis: Implemented clustering (KMeans with Elbow method), regression analysis, and correlation heatmaps to uncover statistical relationships.
  • Step F – Conclusions: Summarized insights such as global trends in COVID and coffee supply, correlation between country activity and economic behavior, and significant outliers by region and year.
